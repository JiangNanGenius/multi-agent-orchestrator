# 📂 案例 / Examples

真实的端到端使用案例，展示多中心协作与专业执行组处理完整任务的全流程。若你尚未完成部署，请先按 `../docs/opencode-ai-deployment-runbook.md` 让 AI 或 opencode 完成正式安装、启动与自检，再回来看这些案例。

| # | 案例 | 任务内容 | 涉及角色 | 复杂度 |
|---|------|---------|----------|--------|
| 1 | [竞品分析](competitive-analysis.md) | "分析 CrewAI vs AutoGen vs LangGraph" | 规划中心→评审中心→数据专家+代码专家+文档专家 | ⭐⭐⭐ |
| 2 | [代码审查](code-review.md) | "审查 FastAPI 代码的安全性" | 规划中心→评审中心→代码专家+审计专家 | ⭐⭐ |
| 3 | [周报生成](weekly-report.md) | "生成本周工程团队周报" | 规划中心→评审中心→数据专家+文档专家 | ⭐⭐ |

每个案例包含：
- **原始任务**：用户输入与目标背景
- **规划中心方案**：任务拆解与执行路径
- **评审中心意见**：审核意见与修订建议
- **专家执行结果**：各专业角色的产出摘要
- **最终交付**：调度中心汇总后的完整结果

> 💡 这些案例基于真实运行记录整理，任务 ID 和时间戳已脱敏。
